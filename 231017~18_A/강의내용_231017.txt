*jQuery : JS 외부 플러그인 라이브러리 소스

- 순수 바닐라 JS 코드로 코딩했을 때보다 현저하게 코딩 난이도가 쉽다. 코딩 개발 노력 덜해도 된다 코딩 글자수도 적다. (*JS < jQuery 웹퍼블리싱 / 프론트엔드 개발 / UI디자인 효율성)

- 부작용 : jQuery : JS 외부 플러그인 => 이들만의 고유 문법 체계를 가지고 있다. 그렇기 때문에 개발자의 창의적인 커스터마이징을 펼치려면 많은 장애요소를 극복해야한다 / 1개의 프로젝트 다수의 jQuery : JS 외부 플러그인 (*플러그인 사이에 충돌)

- 클라이언트 측에서 jQuery : JS 외부 플러그인을 애초에 희망하지 않을 수도 있음

- jQuery : JS 외부 플러그인 => 반드시 협업하는 개발자 혹은 UI디자이너, 퍼블리셔들과 사전 동의!!!

*플러그인 종류
- ScrollTo.js : a태그를 클릭했을 때, 내부에서 페이지 이동 시 부드럽게 이동시키는 효과

- Slick.js : 페이지 내 슬라이더를 생성하고 관리해주는 효과

- Featherlight.js : 페이지 내 모달창을 만들고 관리해주는 효과

- ScrollEffect : 마우스스크롤 시, 페이지 전환효과
