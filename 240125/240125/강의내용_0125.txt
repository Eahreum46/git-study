1) 사용자가 버튼을 클릭한다 (*이벤트)

2) 컴퓨터는 사용자가 본인이 태어난 년도를 입력할 수 있는 창을 제시한다.
- prompt( ) : 사용자가 어떤 값을 입력할 수 있도록 해주는 함수

3) 사용자는 컴퓨터가 제공한 창에다가 본인의 나이를 입력한다.

4) 컴퓨터는 사용자가 입력한 년도를 확인한다

5) 확인한 년도와 현 시점의 년도의 차이를 "-"연산자를 통해서 결과를 도출한다
= 현 시점의 년도를 "변수" 바구니에 담아놯야만 "-" 연산
= 현 시점의 년도 - 사용자가 입력한 년도 결과값을 변수 담을 수 있어야 한다

6) 컴퓨터는 사용자가 해당 결과를 확인할 수 있도록 웹 브라우저에 결과를 출력해준다
